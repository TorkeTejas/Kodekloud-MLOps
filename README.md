# 🚀 100 Days of MLOps

A daily, hands-on journey through the **100 Days of MLOps Challenge** — building
real MLOps skills one task at a time: Linux & servers, environments, dependency
management, experiment tracking, containers, CI/CD, and more.

For each day I include:
- ✅ the completed task
- 📝 plain-language notes explaining the *concepts* behind it (not just the commands)
- 🔍 the reasoning/debugging steps I used to get there

> ⚠️ **Caution — learn it the hard way:**
> I work through each task *myself first*. Hints and solutions come **only after
> genuinely attempting it** — the goal is understanding, not copy-paste.
